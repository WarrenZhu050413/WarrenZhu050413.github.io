Implementation Notes
YouTube Thumbnails (free, no API):
https://img.youtube.com/vi/{VIDEO_ID}/maxresdefault.jpg
Also: hqdefault.jpg, mqdefault.jpg, sddefault.jpg
Vimeo Thumbnails:
Requires oEmbed API call: https://vimeo.com/api/oembed.json?url={VIDEO_URL}
arXiv Paper Preview:
First figure: https://arxiv.org/html/{PAPER_ID}/extracted/.../figures/...
Or use thumbnail service like arxiv-vanity.com
PDF First Page:
Option 1: Pre-generate screenshots with pdftoppm or pdf2image
Option 2: Use PDF.js for client-side rendering
Option 3: Embed with <iframe src="file.pdf#page=1">
